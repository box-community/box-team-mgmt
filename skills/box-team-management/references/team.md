# Team Management

Team members are tracked in a simple `TEAM.md`.

## Format & Template

When creating a new TEAM.md, use this exact template (without example tasks):

```markdown
# Team

## Members

| Name | Email | Role | Location |
| ---- | ----- | ---- | -------- |
|      |       |      |          |

## Current Tasks
```

## Team section

Store team data in a Markdown table.

## Tasks Section

Create a subheader for every team member with tasks, along with a copy of that task and its state
